# Trattoria Verde - Static Website

This is a multi-page static website built with Tailwind CSS, AOS animations, and a small JS include system for header/footer partials.

## Structure

```
site/
├─ partials/
│  ├─ header.html
│  └─ footer.html
├─ css/
│  └─ style.css
├─ js/
│  ├─ include.js
│  └─ main.js
├─ index.html
├─ menu.html
├─ gallery.html
├─ about.html
├─ contact.html
└─ reserve.html
```


## Deployment
### Netlify (recommended for static sites)
1. Create a new site on Netlify.
2. Drag & drop the `site` folder into Netlify (or connect your GitHub repo).
3. Netlify will host it for you and provide HTTPS.
